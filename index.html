Landing Page

    Hi there!

    Welcome to my landing page example

    What is done:

    • Dynamically create a navigation menu based on the sections of the page
    • Scroll to section on link click
    • Distinguish the section in view and set as your-active-class
    • Add an active state to your navigation items when a section is in the viewport
    • Hide fixed navigation bar while not scrolling
    • Make sections collapsible
    • Scroll to top button is added on the page and only visible when the user scrolls below the fold of the page
    • Update/change the design
